# Getting Started with Linux
---

Linux is an operating system ecosystem widely used on servers, cloud infrastructure, data centers, embedded systems, and increasingly on personal computers.  
It is known for stability, security, flexibility, and its open-source nature.

At its core, **Linux itself is not a complete operating system**.  
Linux is the **kernel** used by a family of open-source, Unix-like operating systems, commonly referred to as **GNU/Linux**.

## ğŸ§ Linux Distributions
A **Linux distribution (distro)** is a complete operating system built on top of the Linux kernel.

Each distribution includes:
- The Linux kernel
- GNU tools
- A package manager
- Additional software tailored to specific needs

Examples of user-focused goals:
- Servers (Ubuntu Server, Rocky Linux)
- Desktop users (Ubuntu, Linux Mint)
- Developers and power users (Arch, Fedora)
- Security and penetration testing (Kali Linux)

Different distros exist because different users have different needs.
